Warehouse Automation System

VBA / Automation / WMS Improvement

Project Overview

This project involved improving an existing automated database/table system to improve Warehouse Management System (WMS) workflows. The goal was to increase efficiency, reduce errors, and provide actionable insights for staff and management. To overall close the gap between experience and new workers when it came to analyzing spread sheets for inventory. This is done by completely combining variables from multiple tables into one spread sheet eliminating the need to read multiple tables.
